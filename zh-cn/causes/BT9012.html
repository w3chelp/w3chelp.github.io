BT9012: IE 的 external 对象提供的方法是 IE 特有的

  • 标准参考•
  • 问题描述•
  • 造成的影响•
  • 受影响的浏览器•
  • 问题分析•
  • 解决方案•
  • 参见
作者：孙东国

标准参考

无

问题描述

external 对象提供了对 IE 的一些组件的访问接口，如加入收藏、添加搜索提供商等

造成的影响

该问题将造成某些使用本特性设计的功能在非 IE 浏览器中不能实现。

受影响的浏览器

IE6 IE7 IE8

问题分析

external 对象提供了对 IE 的一些组件的访问接口，这些接口是仅为 IE 设计的，因此其他浏览器不支持。1

关于 external 的更多信息，请参考 MSDN external Object 中的内容。

注：
1. Chrome 也提供了 external 对象，但与 IE 的 external 对象并不相同。

解决方案

不要使用 IE 的特性设计页面代码。
